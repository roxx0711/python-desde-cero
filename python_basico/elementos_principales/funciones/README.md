# Funciones 

Las funciones son bloques de código reutilizables que realizan una tarea específica. Su principal objetivo es evitar repetir código y hacer que los programas sean más organizados, claros y fáciles de mantener.

def sumar(a, b):
    return a + b

resultado = sumar(10, 5)
print("La suma es:", resultado)

## Explicación

>def sumar(a, b): crea una función llamada sumar.
>a y b son los parámetros que recibe la función.
>return a + b devuelve el resultado de la suma.
>sumar(10, 5) llama a la función enviándole los valores 10 y 5.
>print() muestra el resultado en la pantalla.
