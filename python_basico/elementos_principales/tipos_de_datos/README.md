# Tipos de datos en Python 

Son una característica fundamental de la programación que permite clasificar la información almacenada en una variable. Cada tipo de dato define la naturaleza del valor que contiene, como números, texto o valores lógicos, y determina las operaciones que se pueden realizar con él. En Python, los tipos de datos facilitan el manejo y procesamiento correcto de la información dentro de un programa.


>Tipo de dato	        Descripción	                    Ejemplo
>int	               Números enteros.	                  10

>float	               Números decimales.	              3.14

>str	               Cadenas de texto.	              "Hola"

>bool	               Valores lógicos (True o False).	  True

>list	               Colección ordenada y 
>                      modificable de elementos.	      [1, 2, 3]

>tuple	               Colección ordenada que 
>                      no puede modificarse.	          (1, 2, 3)

>set	               Colección desordenada de 
>                      elementos únicos.	               {1, 2, 3}

>dict	               Colección de datos en 
>                      pares clave-valor.	               {"nombre": "Ana", "edad": 20}
>
