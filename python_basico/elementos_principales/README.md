# ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero que permite dar formato a un texto de forma sencilla utilizando símbolos especiales. Se utiliza principalmente para crear archivos README.md.

---

## Símbolos de Markdown

| Símbolo     | Función              | Ejemplo                 |
|-------------|----------------------|-------------------------|
| `#`         | Título principal     | `# Mi proyecto`         |
| `##`        | Subtítulo            | `## Instalación`        |
| `**texto**` | Texto en negrita     | `**Python**`            |
| `*texto*`   | Texto en cursiva     | `*Hola*`                |
| `-`         | Lista                | `- Elemento`            |
| `1.`        | Lista numerada       | `1. Paso uno`           |
| ``` ```      | Bloque de código     | ```python print("Hola")``` |
| `>`         | Cita o nota          | `> Este es un ejemplo.` |

---

## Ejemplo de documentación de código

# Mi primer programa en Python

Este programa pide tu nombre y luego te saluda.

## Código

```python
nombre = input("Escribe tu nombre: ")
print("Hola", nombre)
```

## Explicación

- `input()` pide datos al usuario  
- `print()` muestra el resultado en pantalla  
- `nombre` guarda lo que escribe el usuario
