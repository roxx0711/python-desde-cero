# ¿Que son los BUCLES?

Los bucles son bloques de codigo que repiten una o varias veces instrucciones de forma automatica, ya sea un numero determinado de veces hasta que se cumpla una condición.

Python ocupa dos tipo de BUCLES principales:

## FOR 
Se utiliza cuando se conocer la cantidad de veces que se desea repetir una acción o para recorrer elemento de una lista, una cadena de texto o cualquier colección 

# Ejemplo

frutas = ["manzana", "banana", "cereza"]

>  for fruta in frutas:
>  print(fruta) 

### WHILE
Se utiliza cuando no se sabe cuantas veces se repite el cogido y este continua ejecutandose miestras iuna condicion sea verdadera

# Ejemplo 

contador = 1

>  while contador <= 5:
>     print(contador)
>     contador += 1
