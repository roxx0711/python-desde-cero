# Problemas de práctica: Recursividad y cálculo matemático básico en Python

---

# Parte 1: Problemas con recursividad

---

## Problema 1: Contar desde un número hasta cero

Desarrolla un programa que reciba un número e imprima una cuenta regresiva utilizando una función recursiva.

Ejemplo:

```
Número ingresado:

5

Resultado:

5
4
3
2
1
0
```

---

## Problema 2: Sumar números pares

Desarrolla un programa que calcule la suma de los números pares desde 1 hasta un número ingresado utilizando recursividad.

Ejemplo:

```
Número:

10

Operación:

2 + 4 + 6 + 8 + 10

Resultado:

30
```

---

## Problema 3: Multiplicar números hasta N

Desarrolla un programa que multiplique todos los números desde 1 hasta un número ingresado utilizando una función recursiva.

Ejemplo:

```
Número:

5

Operación:

1 * 2 * 3 * 4 * 5

Resultado:

120
```

---

## Problema 4: Contar números positivos

Desarrolla un programa que reciba un número y cuente cuántos números positivos existen desde ese número hasta 1 utilizando recursividad.

Ejemplo:

```
Número:

5

Resultado:

5 números positivos
```

---

## Problema 5: Sumar los dígitos de un número

Desarrolla un programa que sume cada uno de los dígitos de un número utilizando una función recursiva.

Ejemplo:

```
Número:

1234

Operación:

1 + 2 + 3 + 4

Resultado:

10
```

# Parte 2: Problemas de cálculo matemático básico

---

## Problema 1: Calcular el doble y triple de un número

Desarrolla un programa que solicite un número y calcule:

- El doble.
- El triple.

Ejemplo:

```
Número:

5

Resultado:

Doble: 10

Triple: 15
```

---

## Problema 2: Calcular suma, resta y multiplicación

Desarrolla un programa que solicite dos números y calcule:

- Suma.
- Resta.
- Multiplicación.

Ejemplo:

```
Número 1:

8

Número 2:

3


Resultado:

Suma: 11

Resta: 5

Multiplicación: 24
```

---

## Problema 3: Calcular área de un rectángulo

Desarrolla un programa que solicite la base y altura de un rectángulo.

Debe calcular:

```
Área = base * altura
```

Ejemplo:

```
Base:

5

Altura:

4


Resultado:

Área = 20
```

---

## Problema 4: Convertir grados Celsius a Fahrenheit

Desarrolla un programa que convierta una temperatura Celsius a Fahrenheit.

Fórmula:

```
Fahrenheit = (Celsius * 9/5) + 32
```

Ejemplo:

```
Celsius:

20


Resultado:

68 Fahrenheit
```

---

## Problema 5: Calcular descuento de un producto

Desarrolla un programa que calcule el precio final de un producto aplicando un descuento.

Debe solicitar:

- Precio del producto.
- Porcentaje de descuento.

Ejemplo:

```
Precio:

10000

Descuento:

20%


Resultado:

Precio final:

8000
```
