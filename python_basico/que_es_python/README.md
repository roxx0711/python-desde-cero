# 🐍 ¿Qué es Python?

**Python** es un lenguaje de programación de **alto nivel**, **interpretado** y de **propósito general**, creado para desarrollar programas de manera sencilla, clara y eficiente.

Se caracteriza por tener una **sintaxis simple y fácil de leer**, lo que facilita el aprendizaje tanto para principiantes como para desarrolladores experimentados. Gracias a su versatilidad, es uno de los lenguajes de programación más utilizados en el mundo.

---

## 🤔 ¿Por qué se llama Python?

Aunque muchas personas piensan que el nombre proviene de la serpiente pitón, en realidad **Python recibió su nombre del programa de comedia británico _Monty Python's Flying Circus_**.

Su creador, **Guido van Rossum**, era admirador de este programa y buscaba un nombre corto, fácil de recordar y diferente a los nombres técnicos que solían tener otros lenguajes de programación.

---

## 📌 ¿Qué significa que sea un lenguaje de alto nivel?

Un **lenguaje de alto nivel** es aquel cuya sintaxis es más cercana al lenguaje humano que al lenguaje de las computadoras.

Esto permite que los programadores escriban código de manera más sencilla, sin preocuparse por los detalles internos del hardware.

Por ejemplo, para mostrar un mensaje en pantalla solo se necesita una línea de código:

```python
print("Hola Mundo")
```

---

## ⚙️ ¿Qué significa que sea interpretado?

Python es un lenguaje **interpretado**, lo que significa que el código se ejecuta línea por línea mediante un programa llamado **intérprete**, sin necesidad de compilarlo previamente.

Esto facilita realizar pruebas, corregir errores y desarrollar aplicaciones de forma más rápida.

## 💻 Ejemplo

```python
nombre = input("¿Cuál es tu nombre? ")
print("Hola,", nombre)
```

### Explicación

- `input()` solicita al usuario que escriba información desde el teclado.
- El texto ingresado se almacena en la variable `nombre`.
- `print()` muestra un mensaje en la consola utilizando el valor almacenado en la variable.

### Salida de ejemplo

```text
¿Cuál es tu nombre? Alfonso
Hola, Alfonso
```
