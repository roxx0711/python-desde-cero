# ¿Qué es Python?

Python es un lenguaje de programación de alto nivel. Se caracteriza por su sintaxis sencilla y fácil de leer, lo que facilita el aprendizaje y el desarrollo de programas. Es uno de los lenguajes más utilizados en el mundo para crear aplicaciones web, automatizar tareas, analizar datos, desarrollar inteligencia artificial y mucho más.
